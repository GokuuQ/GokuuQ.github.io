讲座 - 海贼王-路飞是左派还是右派？

发表于 2019-11-17 更新于 2023-03-09 分类于 讲座

摘要：综合了进步的左翼意识和比较民粹保守的统治方式？

路飞是左派还是右派？
有没有必要讨论大众作品的左翼右翼的问题？
一般来讲-
左右的不同多体现于经济
左：建立福利国家、分配关系
在日本较为特殊-
左右的不同体现于政治，和平宪法第九条，关于放弃战争权力，左派重视市民团体、地方自治，右派重视中央官僚；右为冷战思维、日美同盟，右为强调经济发展，左强调综合发展，环境保护、人权等等。
漫画常有的主题-如何正当使用暴力？
左翼把放弃战争权看做为人类进步理念的实践，右翼看做是美国拴住日本的方法。

少年漫画的“政治秩序背景”
-有良好政府、政治秩序和治理但面临的问题超出秩序的能力范畴：龙珠、圣斗士
-无秩序的世界：北斗神拳（“废土世界”）
-历史上的无秩序时代：浪客剑心、银魂
-竞技类漫画（“和平时代的战争”）：运动漫画、美食漫画
-有序治理下边缘的灰色/反体制地带：城市猎人、猫眼三姐妹

OP
-详尽的政治制度和治理设定
-不是生存危机，是主动地竞逐
-“混合政体”：海洋和商业文明底色；天龙人、世界政府（会议）、世界政府（海军）
-收编胁从“王下七武海”-对无秩序的守斌
-“法“外之地-1.神秘文明；2.有领土的海盗；3.革命军

天龙人（主体）唯一的永远的反派
合法性来源：血统
政治职能：系统与秩序的合法性来源
经济基础：经济特权阶层（天上金，与日本历史上给京都公家缴纳的税金名称相同）
内部治理：奴隶制

世界政府（会议）
血统与世俗法律
议事机构
成员国缴纳会费
“平等”议事规则，定期会议，轮值主席

世界政府（海军）
法律（“正义”，能力选拔）
日常治理（维护封建治理秩序；清剿海盗；保障世界政府对“海盗代理人”的控制）
税金
军事等级与法治

王下七武海
王权国家：欧洲式的王权政治

进步的问题意识：
草根的、挑战者的立场
对权威和组织的不信任
历史的虚无？
对自然环境、“传统”和“多元主义”（边缘群体）的推崇
对科技发展后果的怀疑
反对世袭特权和人身压迫
反对有组织的滥用暴力
对金钱和物欲的批评态度

维护正统血统王权？正统王权的善治
不可理解的终极的邪恶：天龙人

海军集中所有批判的问题：捍卫特权、科技、高度组织化

从一种进步的左翼意识到比较民粹保守的落脚点（情绪）
